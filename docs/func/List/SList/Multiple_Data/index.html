🔀 SList_M - 单向链表（多种数据类型）

可存储任意类型数据的单向链表，每个数据自带操作函数

📖 概述

SList_M (Single List Multiple Data) 是一种单向链表数据结构，可以存储任意类型的数据。相比 DList_M，SList_M 占用更少的内存（仅需保存 next 指针），但只支持前向遍历。

⚙️ 数据类型定义

// 链表节点
typedef struct Node_M_inSList {
    struct Node_M_inSList* next;  // 后继节点
    Data_M val;                    // 数据（自带操作集）
} Node_M_inSList;

// SList_M 链表
typedef struct {
    Node_M_inSList* head;      // 头节点
    Node_M_inSList* tail;      // 尾节点
    int size;                  // 链表长度
} SList_M;

🚀 快速开始

#include "base.h"
#include "List/SList/Multiple_Data/slist_mdata.h"
#include "Oper/Int_Info/int_info.h"
#include "Oper/String_Info/string_info.h"

int main() {
    // 1. 初始化链表
    SList_M list;
    initMSList(&list);

    // 2. 插入不同类型的数据
    // 插入整数
    int num = 42;
    Data_M data1 = Data_M_OWN(&num, NULL, &Info_Int, 0);
    insertMValAtEndInMSList(&list, data1);

    // 插入字符串
    char str[] = "Hello";
    Data_M data2 = Data_M_OWN(str, NULL, &Info_String, 0);
    insertMValAtEndInMSList(&list, data2);

    // 3. 打印链表
    printMSList(&list);

    // 4. 释放链表
    freeMSList(&list);
    return 0;
}

💡 完整示例

#include <stdio.h>
#include 
#include "base.h"
#include "List/SList/Multiple_Data/slist_mdata.h"
#include "Oper/Int_Info/int_info.h"
#include "Oper/Double_Info/double_info.h"

int main() {
    // 初始化链表
    SList_M list;
    initMSList(&list);

    // 插入不同类型的数据
    int num1 = 100, num2 = 200;
    double val = 3.14;

    // 插入整数
    Data_M data1 = Data_M_OWN(&num1, NULL, &Info_Int, 0);
    insertMValAtEndInMSList(&list, data1);

    // 插入整数
    Data_M data2 = Data_M_OWN(&num2, NULL, &Info_Int, 0);
    insertMValAtStartInMSList(&list, data2);

    // 插入浮点数
    Data_M data3 = Data_M_OWN(&val, NULL, &Info_Double, 0);
    insertMValAtPosInMSList(&list, data3, 1);

    printf("=== 链表内容 ===\n");
    printMSList(&list);

    // 释放链表
    freeMSList(&list);
    return 0;
}

⚠️ 注意事项

  • SList_M 不需要传入统一的操作集，每个数据都自带 InfoOfData
  • 使用 OWN 模式插入数据时，链表会自动复制一份数据
  • 使用 REF 模式插入数据时，链表只保存指针
  • 通过 getCopy 系列函数获取的数据使用后必须释放
  • 每次使用完链表后必须调用 freeMSList 释放整个链表
  • SList 不支持反向遍历，如果需要反向遍历请使用 DList
