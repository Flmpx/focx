🔗 ChainMap_S - 链式哈希表（单一数据类型）

采用链表法处理冲突的哈希表，存储单一类型的键值对

📖 概述

ChainMap_S (Chain Map Single Data) 是一种链式哈希表数据结构，采用链表法处理哈希冲突。它只能存储单一类型的键和值，需要提供键和值的数据操作集 InfoOfData。

⚙️ 数据类型定义

// 条目类型
typedef struct Entry_S_inChainMap {
    Data_S key;      // 键
    Data_S val;      // 值
    bool isEmpty;    // 是否为空
} Entry_S_inChainMap;

// 链表节点
typedef struct Node_S_inChainMap {
    struct Node_S_inChainMap* next;
    struct Node_S_inChainMap* prev;
    Entry_S_inChainMap entry;
} Node_S_inChainMap;

// 链表
typedef struct List_S_inChainMap {
    Node_S_inChainMap* head;
    Node_S_inChainMap* tail;
    int size;
} List_S_inChainMap;

// ChainMap_S 哈希表
typedef struct ChainMap_S {
    List_S_inChainMap* arr;    // 哈希桶数组
    InfoOfData* keyInfo;       // 键的数据操作集
    InfoOfData* valInfo;       // 值的数据操作集
    int mod;                   // 哈希模数
    int len;                   // 哈希表长度
    int size;                  // 元素数量
} ChainMap_S;

🚀 快速开始

#include "base.h"
#include "Map/ChainMap/Single_Data/chainmap_sdata.h"
#include "Oper/Int_Info/int_info.h"
#include "Oper/String_Info/string_info.h"

int main() {
    // 1. 初始化哈希表
    ChainMap_S map;
    initSChainMap(&map, &Info_Int, &Info_String);

    // 2. 插入键值对
    char* name1 = "Alice";
    char* name2 = "Bob";
    insertSKeyAndSValInSChainMap(&map, 
        Data_S_OWN(1, NULL), Data_Copy, 
        Data_S_OWN(name1, NULL), Data_Copy);
    insertSKeyAndSValInSChainMap(&map, 
        Data_S_OWN(2, NULL), Data_Copy, 
        Data_S_OWN(name2, NULL), Data_Copy);

    // 3. 释放哈希表
    freeSChainMap(&map);
    return 0;
}

🔍 查询操作

Data_S getSKeyBySKeyInSChainMap(ChainMap_S* pMap, Data_S key, selectOfCopy isCopyKey)

通过键找到单类型链式哈希表中的键

参数:
  • pMap - 哈希表指针
  • key - 待查找的键
  • isCopyKey - 返回的键是否要复制
返回: 返回找到的键，若没找到，返回空键（用字段 isEmpty 检查）
说明: 如果不复制，可用于修改内部数据；如果复制，会创建副本，副本默认拥有数据所有权，使用完后通过 freeSKeyInSChainMap 函数释放

说明: 如果不复制，可用于修改内部数据；如果复制，会创建副本，副本默认拥有数据所有权，使用完后通过 freeSValInSChainMap 函数释放

⚠️ 注意事项

  • ChainMap 采用链表法处理哈希冲突，支持更多元素
  • 使用 Data_Copy 模式插入数据时，哈希表会自动复制一份数据
  • 每次使用完哈希表后必须调用 freeSChainMap 释放整个哈希表
