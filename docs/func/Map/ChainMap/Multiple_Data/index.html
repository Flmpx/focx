🔗 ChainMap_M - 链式哈希表（多数据类型）

采用链表法处理冲突的哈希表，存储任意类型的键值对

📖 概述

ChainMap_M (Chain Map Multiple Data) 是一种链式哈希表数据结构，采用链表法处理哈希冲突。它可以存储任意类型的键和值，每条键值对都自带 InfoOfData 操作集指针。

⚙️ 数据类型定义

// 条目类型
typedef struct Entry_M_inChainMap {
    Data_M key;      // 键
    Data_M val;      // 值
    bool isEmpty;    // 是否为空
} Entry_M_inChainMap;

// 链表节点
typedef struct Node_M_inChainMap {
    struct Node_M_inChainMap* next;
    struct Node_M_inChainMap* prev;
    Entry_M_inChainMap entry;
} Node_M_inChainMap;

// 链表
typedef struct List_M_inChainMap {
    Node_M_inChainMap* head;
    Node_M_inChainMap* tail;
    int size;
} List_M_inChainMap;

// ChainMap_M 哈希表
typedef struct ChainMap_M {
    List_M_inChainMap* arr;    // 哈希桶数组
    int mod;                   // 哈希模数
    int len;                   // 哈希表长度
    int size;                  // 元素数量
} ChainMap_M;

🚀 快速开始

#include "base.h"
#include "Map/ChainMap/Multiple_Data/chainmap_mdata.h"
#include "Oper/Int_Info/int_info.h"
#include "Oper/String_Info/string_info.h"

int main() {
    // 1. 初始化哈希表
    ChainMap_M map;
    initMChainMap(&map);

    // 2. 插入不同类型的键值对
    int id = 1;
    double score = 95.5;
    
    insertMKeyAndMValInMChainMap(&map, 
        Data_M_OWN(&id, NULL, &Info_Int, TYPE_INT), Data_Copy,
        Data_M_OWN(&score, NULL, &Info_Double, TYPE_DOUBLE), Data_Copy);

    // 3. 释放哈希表
    freeMChainMap(&map);
    return 0;
}

➕ 插入操作

InfoOfReturn insertMKeyAndMValInMChainMap(ChainMap_M* pMap, Data_M key, selectOfCopy isCopyKey, Data_M val, selectOfCopy isCopyVal)

在多类型链式哈希表中插入键值对

参数:
  • pMap - 哈希表指针
  • key - 待插入的键
  • isCopyKey - 是否要复制传入的 key
  • val - 待插入的值
  • isCopyVal - 是否要复制传入的 val
返回: 操作结果状态码

🔍 查询操作

Data_M getMKeyByMKeyInMChainMap(ChainMap_M* pMap, Data_M key, selectOfCopy isCopyKey)

通过键找到多类型链式哈希表中的键

参数:
  • pMap - 哈希表指针
  • key - 待查找的键
  • isCopyKey - 返回的键是否要复制
返回: 返回找到的键，若没找到，返回空键（用字段 isEmpty 检查）
说明: 如果不复制，可用于修改内部数据；如果复制，会创建副本，副本默认拥有数据所有权，使用完后通过 freeMKeyInMChainMap 函数释放

通过键找到多类型链式哈希表中的值

参数:
  • pMap - 哈希表指针
  • key - 待查找的值的键
  • isCopyVal - 返回的值是否要复制
返回: 返回找到的值，若没找到，返回空值（用字段 isEmpty 检查）
说明: 如果不复制，可用于修改内部数据；如果复制，会创建副本，副本默认拥有数据所有权，使用完后通过 freeMValInMChainMap 函数释放

说明: 如果不复制，可用于修改内部数据；如果复制，会创建副本，副本默认拥有数据所有权，使用完后通过 freeMEntryInMChainMap 函数释放

⚠️ 注意事项

  • ChainMap_M 每条键值对都自带 InfoOfData，可以存储多种不同类型的键值对
  • ChainMap 采用链表法处理哈希冲突，支持更多元素
  • 每次使用完哈希表后必须调用 freeMChainMap 释放整个哈希表
