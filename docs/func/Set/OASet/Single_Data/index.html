✴️ OASet_S - 开放地址哈希集合（单一数据类型）

采用开放定址法解决哈希冲突的集合存储，需要提供键的数据操作集

📖 概述

OASet_S (Open Addressing Set Single Data) 采用开放定址法（Open Addressing）解决哈希冲突。当发生冲突时，使用线性探测（Linear Probing）寻找下一个可用的槽位。

⚙️ 数据类型定义

// 集合条目
typedef struct Entry_S_inOASet {
    Data_S key;      // 键
    int state;       // 状态
    bool isEmpty;    // 是否为空
} Entry_S_inOASet;

// 哈希集合
typedef struct OASet_S {
    Entry_S_inOASet* arr;    // 哈希表数组
    int mod;                 // 模数
    int len;                 // 当前元素数量
    int size;                // 哈希表容量
    InfoOfData* keyInfo;     // 键的操作集
} OASet_S;

🚀 快速开始

#include "base.h"
#include "Set/OASet/Single_Data/oaset_sdata.h"
#include "Oper/Int_Info/int_info.h"

int main() {
    // 1. 初始化集合
    OASet_S set;
    initSOASet(&set, &Info_Int);

    // 2. 插入键
    insertSKeyInSOASet(&set, Data_S_OWN(1, NULL));
    insertSKeyInSOASet(&set, Data_S_OWN(2, NULL));
    insertSKeyInSOASet(&set, Data_S_OWN(3, NULL));

    // 3. 检查键是否存在
    if (hasSKeyInSOASet(&set, Data_S_OWN(2, NULL))) {
        printf("Key 2 exists!\n");
    }

    // 4. 删除键
    delSKeyBySKeyInSOASet(&set, Data_S_OWN(2, NULL));

    // 5. 打印集合
    printSOASet(&set);

    // 6. 释放集合
    freeSOASet(&set);
    return 0;
}

🔍 查询操作

Data_S getCopySKeyBySKeyInSOASet(OASet_S* pSet, Data_S key)

通过键找到单类型开放寻址集合中的键，使用完后用 freeSKeyInSOASet 函数进行释放

参数:
  • pSet - 集合指针
  • key - 待查找的键
返回: 返回找到的键，若没找到，返回空键（用字段 isEmpty 检查）
警告: 返回的是复制的数据，使用后需调用 freeSKeyInSOASet 释放

返回: 如果存在返回 true，否则 false

⚠️ 注意事项

  • Set 只存储唯一键，不允许重复插入
  • 通过 getCopy 系列函数获取的数据使用后必须释放
  • 每次使用完集合后必须调用 freeSOASet 释放整个集合
  • OASet 使用开放定址法，适合数据量较小且分布均匀的场景
