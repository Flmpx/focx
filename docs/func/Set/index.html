🎯 Set 集合

哈希集合数据结构，用于存储唯一键

📖 概述

Set 模块提供哈希集合实现，用于存储不重复的键。支持高效的成员检测、插入和删除操作。

📌 可用类型

  • ChainSet - 链表法哈希集合（适合大数据量）
  • OASet - 开放定址法哈希集合（适合小数据量）

🗂️ 目录结构

ChainSet 链表法

采用链表法解决哈希冲突，适合数据量较大且分布不均匀的场景

ChainSet_S 单一数据类型 ChainSet_M 多种数据类型

OASet 开放定址法

采用开放定址法解决哈希冲突，内存利用率高，适合小数据量

OASet_S 单一数据类型 OASet_M 多种数据类型

💡 选择指南

特性 ChainSet OASet
冲突处理 链表法 开放定址法
内存使用 较高（含指针） 较低（连续数组）
数据量 适合大数据量 适合小数据量
查找性能 较稳定 可能退化
推荐场景 分布不均匀 分布均匀
