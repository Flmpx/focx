🔐 ChainSet_S - 链式哈希集合（单一数据类型）

采用链表法解决哈希冲突的集合存储，需要提供键的数据操作集

📖 概述

ChainSet_S (Chain Set Single Data) 采用链表法（Separate Chaining）解决哈希冲突。每个哈希桶都是一个链表，可以在同义词冲突时将多个键链接在一起。

⚙️ 数据类型定义

// 集合条目
typedef struct Entry_S_inChainSet {
    Data_S key;      // 键
    bool isEmpty;    // 是否为空
} Entry_S_inChainSet;

// 哈希集合
typedef struct ChainSet_S {
    List_S_inChainSet* arr;    // 哈希桶数组
    InfoOfData* keyInfo;       // 键的操作集
    int mod;                   // 模数
    int len;                   // 当前元素数量
    int size;                  // 哈希表容量
} ChainSet_S;

🚀 快速开始

#include "base.h"
#include "Set/ChainSet/Single_Data/chainset_sdata.h"
#include "Oper/Int_Info/int_info.h"

int main() {
    // 1. 初始化集合
    ChainSet_S set;
    initSChainSet(&set, &Info_Int);

    // 2. 插入键
    insertSKeyInSChainSet(&set, Data_S_OWN(1, NULL));
    insertSKeyInSChainSet(&set, Data_S_OWN(2, NULL));
    insertSKeyInSChainSet(&set, Data_S_OWN(3, NULL));

    // 3. 检查键是否存在
    if (hasSKeyInSChainSet(&set, Data_S_OWN(2, NULL))) {
        printf("Key 2 exists!\n");
    }

    // 4. 删除键
    delSKeyBySKeyInSChainSet(&set, Data_S_OWN(2, NULL));

    // 5. 打印集合
    printSChainSet(&set);

    // 6. 释放集合
    freeSChainSet(&set);
    return 0;
}

📋 函数列表

🔧 初始化与销毁

void initSChainSet(ChainSet_S* pSet, InfoOfData* keyInfo)

初始化单类型链式集合

参数:
  • pSet - 集合指针
  • keyInfo - 键的数据信息的指针
void freeSKeyInSChainSet(ChainSet_S* pSet, Data_S* key)

释放单类型链式集合中的键

参数:
  • pSet - 集合指针
  • key - 待释放的键
void freeSChainSet(ChainSet_S* pSet)

释放单类型链式集合

参数:
  • pSet - 集合指针

➕ 插入操作

InfoOfReturn insertSKeyInSChainSet(ChainSet_S* pSet, Data_S key)

在单类型链式集合中插入键

参数:
  • pSet - 集合指针
  • key - 待插入的键
返回: 操作结果状态码

🔍 查询操作

Data_S getCopySKeyBySKeyInSChainSet(ChainSet_S* pSet, Data_S key)

通过键找到单类型链式集合中的键，使用完后用 freeSKeyInSChainSet 函数进行释放

参数:
  • pSet - 集合指针
  • key - 待查找的键
返回: 返回找到的键，若没找到，返回空键（用字段 isEmpty 检查）
警告: 返回的是复制的数据，使用后需调用 freeSKeyInSChainSet 释放
bool hasSKeyInSChainSet(ChainSet_S* pSet, Data_S key)

判断单类型链式集合中是否存在当前键

参数:
  • pSet - 集合指针
  • key - 待查找的键
返回: 如果存在返回 true，否则 false

➖ 删除操作

InfoOfReturn delSKeyBySKeyInSChainSet(ChainSet_S* pSet, Data_S key)

删除单类型链式集合中的键

参数:
  • pSet - 集合指针
  • key - 待删除的键
返回: 操作结果状态码

打印单类型链式集合的键

参数:
  • pSet - 集合指针
  • key - 待打印的键
void printSChainSet(ChainSet_S* pSet)

打印单类型链式集合

参数:
  • pSet - 集合指针

⚠️ 注意事项

  • Set 只存储唯一键，不允许重复插入
  • 通过 getCopy 系列函数获取的数据使用后必须释放
  • 每次使用完集合后必须调用 freeSChainSet 释放整个集合
  • ChainSet 使用链表法解决冲突，适合数据量较大或分布不均匀的场景
