📍 OASet_S - 开放寻址集合（单一数据类型）

采用开放寻址法处理冲突的哈希集合，存储单一类型的元素

📖 概述

OASet_S (Open Addressing Set Single Data) 是一种开放寻址哈希集合数据结构，采用开放寻址法处理哈希冲突。它只能存储单一类型的元素，需要提供数据操作集 InfoOfData。

⚙️ 数据类型定义

// 条目类型
typedef struct Entry_S_inOASet {
    Data_S key;      // 键
    int state;       // 状态（用于标记删除或占用）
    bool isEmpty;    // 是否为空
} Entry_S_inOASet;

// OASet_S 集合
typedef struct OASet_S {
    Entry_S_inOASet* arr;    // 哈希桶数组
    int mod;                 // 哈希模数
    int len;                 // 哈希表长度
    int size;                // 元素数量
    InfoOfData* keyInfo;     // 键的数据操作集
} OASet_S;

🚀 快速开始

#include "base.h"
#include "Set/OASet/Single_Data/oaset_sdata.h"
#include "Oper/Int_Info/int_info.h"

int main() {
    // 1. 初始化集合
    OASet_S set;
    initSOASet(&set, &Info_Int);

    // 2. 插入元素
    int i;
    for (i = 0; i < 10; i++) {
        insertSKeyInSOASet(&set, Data_S_OWN(&i, NULL), Data_Copy);
    }

    // 3. 释放集合
    freeSOASet(&set);
    return 0;
}

📋 函数列表

🔧 初始化与销毁

void initSOASet(OASet_S* pSet, InfoOfData* keyInfo)

初始化单类型开放寻址集合

参数:
  • pSet - 集合指针
  • keyInfo - 键的数据信息的指针
void freeSKeyInSOASet(OASet_S* pSet, Data_S* key)

释放单类型开放寻址集合中的键

参数:
  • pSet - 集合指针
  • key - 待释放的键
void freeSOASet(OASet_S* pSet)

释放单类型开放寻址集合

参数:
  • pSet - 集合指针

➕ 插入操作

InfoOfReturn insertSKeyInSOASet(OASet_S* pSet, Data_S key, selectOfCopy isCopyKey)

在单类型开放寻址集合中插入键

参数:
  • pSet - 集合指针
  • key - 待插入的键
  • isCopyKey - 是否要复制传入的 key
返回: 操作结果状态码

🔍 查询操作

Data_S getSKeyBySKeyInSOASet(OASet_S* pSet, Data_S key, selectOfCopy isCopyKey)

通过键找到单类型开放寻址集合中的键

参数:
  • pSet - 集合指针
  • key - 待查找的键
  • isCopyKey - 返回的键是否要复制
返回: 返回找到的键，若没找到，返回空键（用字段 isEmpty 检查）
说明: 如果不复制，可用于修改内部数据；如果复制，会创建副本，副本默认拥有数据所有权，使用完后通过 freeSKeyInSOASet 函数释放
bool hasSKeyInSOASet(OASet_S* pSet, Data_S key)

判断单类型开放寻址集合中是否存在当前键

参数:
  • pSet - 集合指针
  • key - 待查找的键
返回: 如果存在返回 true，否则 false

➖ 删除操作

InfoOfReturn delSKeyBySKeyInSOASet(OASet_S* pSet, Data_S key)

删除单类型开放寻址集合中的键

参数:
  • pSet - 集合指针
  • key - 待删除的键
返回: 操作结果状态码

⚠️ 注意事项

  • Set 中的元素唯一，不允许重复
  • OASet 采用开放寻址法处理哈希冲突，空间利用率高
  • 使用 Data_Copy 模式插入数据时，集合会自动复制一份数据
  • 每次使用完集合后必须调用 freeSOASet 释放整个集合
