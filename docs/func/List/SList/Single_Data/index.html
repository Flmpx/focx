➡️ SList_S - 单向链表（单一数据类型）

存储单一类型数据的单向链表，需要提供数据操作集 (InfoOfData)

📖 概述

SList_S (Single List Single Data) 是一种单向链表数据结构，只能存储单一类型的数据。相比 DList，SList 占用更少的内存（仅需保存 next 指针），但只支持前向遍历。

⚙️ 数据类型定义

// 链表节点
typedef struct Node_S_inSList {
    struct Node_S_inSList* next;  // 后继节点
    Data_S val;                    // 数据
} Node_S_inSList;

// SList_S 链表
typedef struct {
    Node_S_inSList* head;      // 头节点
    Node_S_inSList* tail;      // 尾节点
    InfoOfData* valInfo;       // 数据操作集
    int size;                  // 链表长度
} SList_S;

🚀 快速开始

#include "base.h"
#include "List/SList/Single_Data/slist_sdata.h"
#include "Oper/Int_Info/int_info.h"  // 使用预定义整数操作集

int main() {
    // 1. 初始化链表
    SList_S list;
    initSSList(&list, &Info_Int);

    // 2. 创建数据并插入
    Data_S data1 = Data_S_OWN(123, NULL);
    insertSValAtEndInSSList(&list, data1);

    Data_S data2 = Data_S_OWN(456, NULL);
    insertSValAtEndInSSList(&list, data2);

    // 3. 打印链表
    printSSList(&list);

    // 4. 释放链表
    freeSSList(&list);
    return 0;
}

💡 完整示例

#include <stdio.h>
#include "base.h"
#include "List/SList/Single_Data/slist_sdata.h"
#include "Oper/Int_Info/int_info.h"

int main() {
    // 初始化链表
    SList_S list;
    initSSList(&list, &Info_Int);

    // 插入数据
    printf("=== 插入数据 ===\n");
    insertSValAtEndInSSList(&list, Data_S_OWN(100, NULL));
    insertSValAtEndInSSList(&list, Data_S_OWN(200, NULL));
    insertSValAtStartInSSList(&list, Data_S_OWN(50, NULL));
    insertSValAtPosInSSList(&list, Data_S_OWN(150, NULL), 2);
    printSSList(&list);

    // 查找数据
    printf("\n=== 查找数据 ===\n");
    Data_S found = getPtrSValByPosInSSList(&list, 2);
    if (!found.isEmpty) {
        printf("位置 2 的数据: %d\n", *(int*)found.data);
    }

    // 判断是否存在
    printf("\n是否存在 200: %s\n", hasSValInSSList(&list, Data_S_OWN(200, NULL)) ? "是" : "否");

    // 删除数据
    printf("\n=== 删除数据 ===\n");
    delNodeByPosInSSList(&list, 1);
    printSSList(&list);

    // 释放链表
    freeSSList(&list);
    return 0;
}

⚠️ 注意事项

  • 使用 OWN 模式插入数据时，链表会自动复制一份数据
  • 使用 REF 模式插入数据时，链表只保存指针，不会复制数据
  • 通过 getCopy 系列函数获取的数据使用后必须释放
  • 通过 getPtr 系列函数获取的数据指针直接指向链表内部，不要释放
  • 每次使用完链表后必须调用 freeSSList 释放整个链表
  • SList 不支持反向遍历，如果需要反向遍历请使用 DList
