🌀 ChainSet_M - 链式哈希集合（多种数据类型）

采用链表法解决哈希冲突的集合存储，键可存储任意类型数据

📖 概述

ChainSet_M (Chain Set Multiple Data) 采用链表法（Separate Chaining）解决哈希冲突。每个哈希桶都是一个链表，可以在同义词冲突时将多个键链接在一起。

⚙️ 数据类型定义

// 集合条目
typedef struct Entry_M_inChainSet {
    Data_M key;      // 键
    bool isEmpty;    // 是否为空
} Entry_M_inChainSet;

// 哈希集合
typedef struct ChainSet_M {
    List_M_inChainSet* arr;    // 哈希桶数组
    int mod;                   // 模数
    int len;                   // 当前元素数量
    int size;                  // 哈希表容量
} ChainSet_M;

🚀 快速开始

#include "base.h"
#include "Set/ChainSet/Multiple_Data/chainset_mdata.h"
#include "Oper/Int_Info/int_info.h"
#include "Oper/String_Info/string_info.h"

int main() {
    // 1. 初始化集合
    ChainSet_M set;
    initMChainSet(&set);

    // 2. 插入键
    int num = 1;
    Data_M key1 = Data_M_OWN(&num, NULL, &Info_Int, 0);
    insertMKeyInMChainSet(&set, key1);

    char str[] = "hello";
    Data_M key2 = Data_M_OWN(str, NULL, &Info_String, 0);
    insertMKeyInMChainSet(&set, key2);

    // 3. 打印集合
    printMChainSet(&set);

    // 4. 释放集合
    freeMChainSet(&set);
    return 0;
}

🔍 查询操作

Data_M getCopyMKeyByMKeyInMChainSet(ChainSet_M* pSet, Data_M key)

通过键找到多类型链式集合中的键，使用完后用 freeMKeyInMChainSet 函数进行释放

参数:
  • pSet - 集合指针
  • key - 待查找的键
返回: 返回找到的键，若没找到，返回空键（用字段 isEmpty 检查）
警告: 返回的是复制的数据，使用后需调用 freeMKeyInMChainSet 释放
bool hasMKeyInMChainSet(ChainSet_M* pSet, Data_M key)

参数:
  • pSet - 集合指针
  • key - 待查找的键
返回: 如果存在返回 true，否则 false

⚠️ 注意事项

  • Set 只存储唯一键，不允许重复插入
  • 通过 getCopy 系列函数获取的数据使用后必须释放
  • 每次使用完集合后必须调用 freeMChainSet 释放整个集合
  • ChainSet 使用链表法解决冲突，适合数据量较大或分布不均匀的场景
