📋 List 链表

链表数据结构

📖 概述

List 模块提供链表实现，支持在链表头部、尾部或指定位置插入/删除元素。

📌 可用类型

  • DList - 双向链表（支持前向和后向遍历）
  • SList - 单向链表（内存占用更少）

🗂️ 目录结构

DList - 双向链表

支持前向和后向遍历，可在任意位置高效插入/删除

DList_S 单一数据类型链表 DList_M 多种数据类型链表

SList - 单向链表

仅支持前向遍历，占用更少内存

SList_S 单一数据类型链表 SList_M 多种数据类型链表

💡 选择指南

类型 遍历方向 适用场景 特点
DList 双向 需要反向遍历 占用略高，功能更强
SList 单向 内存敏感场景 占用更低，简单高效

📊 数据类型后缀说明

后缀 数据类型 灵活性 内存开销
_S 统一类型 中等 较低
_M 任意类型 高 较高
