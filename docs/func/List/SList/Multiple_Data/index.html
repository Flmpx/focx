➡️ SList_M - 单向链表（多数据类型）

存储任意类型数据的单向链表，每条数据自带 InfoOfData

📖 概述

SList_M (Single List Multiple Data) 是一种单向链表数据结构，可以存储任意类型的数据。每条数据都是独立的 Data_M 类型，自带 InfoOfData 操作集指针，因此可以同时存储不同类型的数据。

说明: SList 与 DList 的区别在于 SList 是单向的，只能从头向尾遍历，不能反向遍历

⚙️ 数据类型定义

// 链表节点
typedef struct Node_M_inSList {
    struct Node_M_inSList* next;   // 后继节点
    Data_M val;                    // 数据
} Node_M_inSList;

// SList_M 链表
typedef struct {
    Node_M_inSList* head;      // 头节点
    Node_M_inSList* tail;      // 尾节点
    int size;                  // 链表长度
} SList_M;

🚀 快速开始

#include "base.h"
#include "List/SList/Multiple_Data/slist_mdata.h"
#include "Oper/Int_Info/int_info.h"

int main() {
    // 1. 初始化链表
    SList_M list;
    initMSList(&list);

    // 2. 插入不同类型的数据
    int i = 100;
    double d = 3.14;
    
    insertMValAtEndInMSList(&list, Data_M_OWN(&i, NULL, &Info_Int, TYPE_INT), Data_Copy);
    insertMValAtEndInMSList(&list, Data_M_OWN(&d, NULL, &Info_Double, TYPE_DOUBLE), Data_Copy);

    // 3. 释放链表
    freeMSList(&list);
    return 0;
}

⚠️ 注意事项

  • SList 是单向链表，只能从头向尾遍历，不能反向遍历
  • SList_M 每条数据都自带 InfoOfData，可以存储多种不同类型的数据
  • 使用 Data_Copy 模式插入数据时，链表会自动复制一份数据
  • 每次使用完链表后必须调用 freeMSList 释放整个链表
