🔗 ChainSet_S - 链式哈希集合（单一数据类型）

采用链表法处理冲突的哈希集合，存储单一类型的元素

📖 概述

ChainSet_S (Chain Set Single Data) 是一种链式哈希集合数据结构，采用链表法处理哈希冲突。它只能存储单一类型的元素，需要提供数据操作集 InfoOfData。

⚙️ 数据类型定义

// 条目类型
typedef struct Entry_S_inChainSet {
    Data_S key;      // 键
    bool isEmpty;    // 是否为空
} Entry_S_inChainSet;

// 链表节点
typedef struct Node_S_inChainSet {
    struct Node_S_inChainSet* next;
    struct Node_S_inChainSet* prev;
    Entry_S_inChainSet entry;
} Node_S_inChainSet;

// 链表
typedef struct List_S_inChainSet {
    Node_S_inChainSet* head;
    Node_S_inChainSet* tail;
    int size;
} List_S_inChainSet;

// ChainSet_S 集合
typedef struct ChainSet_S {
    List_S_inChainSet* arr;    // 哈希桶数组
    InfoOfData* keyInfo;       // 键的数据操作集
    int mod;                   // 哈希模数
    int len;                   // 哈希表长度
    int size;                  // 元素数量
} ChainSet_S;

🚀 快速开始

#include "base.h"
#include "Set/ChainSet/Single_Data/chainset_sdata.h"
#include "Oper/Int_Info/int_info.h"

int main() {
    // 1. 初始化集合
    ChainSet_S set;
    initSChainSet(&set, &Info_Int);

    // 2. 插入元素
    int i;
    for (i = 0; i < 10; i++) {
        insertSKeyInSChainSet(&set, Data_S_OWN(&i, NULL), Data_Copy);
    }

    // 3. 释放集合
    freeSChainSet(&set);
    return 0;
}

📋 函数列表

🔧 初始化与销毁

void initSChainSet(ChainSet_S* pSet, InfoOfData* keyInfo)

初始化单类型链式集合

参数:
  • pSet - 集合指针
  • keyInfo - 键的数据信息的指针
void freeSKeyInSChainSet(ChainSet_S* pSet, Data_S* key)

释放单类型链式集合中的键

参数:
  • pSet - 集合指针
  • key - 待释放的键
void freeSChainSet(ChainSet_S* pSet)

释放单类型链式集合

参数:
  • pSet - 集合指针

➕ 插入操作

InfoOfReturn insertSKeyInSChainSet(ChainSet_S* pSet, Data_S key, selectOfCopy isCopyKey)

在单类型链式集合中插入键

参数:
  • pSet - 集合指针
  • key - 待插入的键
  • isCopyKey - 是否要复制传入的 key
返回: 操作结果状态码

🔍 查询操作

Data_S getSKeyBySKeyInSChainSet(ChainSet_S* pSet, Data_S key, selectOfCopy isCopyKey)

通过键找到单类型链式集合中的键

参数:
  • pSet - 集合指针
  • key - 待查找的键
  • isCopyKey - 返回的键是否要复制
返回: 返回找到的键，若没找到，返回空键（用字段 isEmpty 检查）
说明: 如果不复制，可用于修改内部数据；如果复制，会创建副本，副本默认拥有数据所有权，使用完后通过 freeSKeyInSChainSet 函数释放
bool hasSKeyInSChainSet(ChainSet_S* pSet, Data_S key)

判断单类型链式集合中是否存在当前键

参数:
  • pSet - 集合指针
  • key - 待查找的键
返回: 如果存在返回 true，否则 false

➖ 删除操作

InfoOfReturn delSKeyBySKeyInSChainSet(ChainSet_S* pSet, Data_S key)

删除单类型链式集合中的键

参数:
  • pSet - 集合指针
  • key - 待删除的键
返回: 操作结果状态码

⚠️ 注意事项

  • Set 中的元素唯一，不允许重复
  • ChainSet 采用链表法处理哈希冲突，支持更多元素
  • 使用 Data_Copy 模式插入数据时，集合会自动复制一份数据
  • 每次使用完集合后必须调用 freeSChainSet 释放整个集合
