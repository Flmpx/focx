🔧 FOCX 函数大全

FOCX 数据结构库的完整 API 参考

📋 List 模块 - 链表

链表数据结构，支持在任意位置插入、删除和查找数据

DList - 双向链表

支持前向和后向遍历

🔗

DList_S - 单数据类型

使用单个 InfoOfData 管理数据
🧬

DList_M - 多数据类型

每个节点可存储多个不同类型的数据

SList - 单向链表

占用更少内存，仅支持前向遍历

➡️

SList_S - 单数据类型

使用单个 InfoOfData 管理数据
🔀

SList_M - 多数据类型

每个节点可存储多个不同类型的数据

🗺️ Map 模块 - 哈希映射表

键值对存储，提供高效的查找、插入和删除操作

ChainMap - 链式哈希表

使用链表解决哈希冲突

🔑

ChainMap_S - 单数据类型

键和值都使用单个 InfoOfData
🗝️

ChainMap_M - 多数据类型

键和值都使用多个 Data_M

OAMap - 开放地址哈希表

使用线性探测解决哈希冲突，占用更少内存

💠

OAMap_S - 单数据类型

键和值都使用单个 InfoOfData
💎

OAMap_M - 多数据类型

键和值都使用多个 Data_M

🎯 Set 模块 - 哈希集合

集合数据结构，存储唯一键，支持高效的成员检测

ChainSet - 链式哈希集合

使用链表解决哈希冲突

🔐

ChainSet_S - 单数据类型

使用单个 InfoOfData 管理键
🌀

ChainSet_M - 多数据类型

每个键可存储多个不同类型的数据

OASet - 开放地址哈希集合

使用线性探测解决哈希冲突，占用更少内存

✴️

OASet_S - 单数据类型

使用单个 InfoOfData 管理键
🔶

OASet_M - 多数据类型

每个键可存储多个不同类型的数据
